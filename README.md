# GravityPlusPlus
A Graph-based Visualisation Software for Interactive Visualisation Design.

![alt text](./assets/images/data-screenshot.png)

Build the docker container.

    docker-compose build
    
Build the client
    
    cd client
    
    npm install 
    
Run the Node server and Neo4J database.

    docker-compose up
    
To run the client, 
    
    cd client
    
    npm start
